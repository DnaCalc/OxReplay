# OxReplay Upstream Notes

Use this directory for outbound notes from `OxReplay` when this repo needs to tell sibling repos or Foundation about:
1. adapter contract implications,
2. source-schema or bundle-shape implications,
3. lifecycle or registry needs,
4. replay-host requirements,
5. known limitations or migration constraints.

Current local note files:
1. `NOTES_FOR_OXCALC.md`
2. `NOTES_FOR_OXFML.md`
3. `NOTES_FOR_OXFUNC.md`
4. `NOTES_FOR_OXVBA.md`
5. `NOTES_FOR_FOUNDATION.md`

Inbound notes to `OxReplay` are expected to live in the sibling repos, for example:
1. `../OxCalc/docs/upstream/NOTES_FOR_OXREPLAY.md`
2. `../OxFml/docs/upstream/NOTES_FOR_OXREPLAY.md`
3. `../OxFunc/docs/upstream/NOTES_FOR_OXREPLAY.md`
