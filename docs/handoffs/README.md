# OxReplay Handoffs

This directory holds structured handoff packets between `OxReplay`, Foundation, and sibling repos.

## Rules
1. Every normative or contract-affecting handoff should be registered in `HANDOFF_REGISTER.csv`.
2. Receipt or acknowledgement packets should be retained alongside the originating handoff when they exist.
3. Handoffs do not close worksets on their own; they record a dependency and evidence trail.
